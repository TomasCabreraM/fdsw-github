Mis trabajos

Plataforma Web para Eventos de Alto Impacto

Desarrollé una Single Page Application para gestionar un evento exclusivo en la Patagonia. Utilicé React y Next.js para una experiencia fluida con reservas en tiempo real. Implementé transmisión en vivo con WebRTC y configuré un despliegue en AWS, garantizando escalabilidad y alta disponibilidad mediante servicios como CloudFront y Elastic Beanstalk.

Ecosistema de Marketplace Global

Lideré la creación de una plataforma web que conecta productores locales con compradores internacionales. El backend fue desarrollado con Node.js y Express, manejando operaciones de usuarios, pedidos y logística. El frontend, creado con Vue.js, ofrece un diseño optimizado para mejorar la navegación y las conversiones. La seguridad se reforzó con OAuth2 y gestión de roles.

Portal de Trazabilidad Digital

Construí un portal web para asegurar la trazabilidad de productos marinos, utilizando Laravel en el backend y MySQL para la base de datos. El sistema incluye paneles administrativos con reportes dinámicos, autenticación con JWT y un frontend responsivo desarrollado con Bootstrap. La integración de APIs externas mejoró la interoperabilidad con los servicios de logística.
